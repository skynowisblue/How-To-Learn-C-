# How To Learn C

A beginner-friendly guide to learning C programming, written by a 9-year-old for other kids.

## How to compile and run

1. Open a terminal / command prompt
2. Navigate to the folder containing the .c file
3. Run: `gcc filename.c -o filename`
4. Run: `./filename`
